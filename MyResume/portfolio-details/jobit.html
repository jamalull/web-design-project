JobIT - Job Portal Web

Project information

  • Category: Web Apps
  • Technology: React.js, and Tailwind CSS.
  • Project URL: Link Demo Apps

JobIT is a job portal web application to facilitate job seekers and companies to find employment agreements. This application has various kinds of job vacancies such as IT, marketing, sales, and others. Equipped with features for logging in and registering user accounts, creating and updating job vacancy posts, and displaying job vacancy information via an API.
